# Amine Codes - Professional Portfolio

Welcome to the official repository for **Amine Codes**, a professional and modern developer portfolio website designed to showcase projects, skills, and certifications. 

## 🌐 Live Demo
*(Add your live deployed link here once deployed, e.g., https://kuswahabhupendra75-debug.github.io/portfolio)*

## ✨ Features
- **Modern & Premium UI:** Crafted using modern design principles with a vibrant, sleek interface.
- **Dark/Light Mode:** Includes a fully functional theme toggle (sun/moon icon) that remembers user preferences via local storage.
- **Responsive Design:** Optimized for desktops, tablets, and mobile devices using Tailwind CSS.
- **GSAP Animations:** Smooth scroll animations and element reveals for an immersive experience.
- **Interactive Typewriter Effect:** Engaging dynamic text in the hero section.
- **Projects Showcase:** Highlight key projects with descriptions, tech stacks, and live links.
- **Certifications Gallery:** A dedicated section displaying professional achievements and verifiable credentials.

## 🛠️ Built With
- **HTML5 & CSS3**
- **JavaScript (Vanilla)**
- **[Tailwind CSS](https://tailwindcss.com/)** (via CDN for rapid styling)
- **[GSAP](https://gsap.com/)** (GreenSock Animation Platform)
- **[Font Awesome](https://fontawesome.com/)** (Icons)

## 🚀 Getting Started

### Prerequisites
You can run this project locally without any complex build tools. A simple local server is recommended.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/kuswahabhupendra75-debug/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Run the project locally:
   - If you have Node.js installed, you can use the provided Windows batch script:
     ```bash
     .\RUN_PORTFOLIO.bat
     ```
   - Alternatively, use any live server extension (like VS Code Live Server) or Python's HTTP server:
     ```bash
     python -m http.server 8000
     ```
